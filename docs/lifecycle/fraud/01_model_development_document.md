# Model Development Document

## 1. Purpose & Intended Use
The purpose of this model is to predict whether a card transaction is fraudulent, enabling real-time authorization decisioning. This model is designed to assist financial institutions in identifying potentially fraudulent transactions, thereby reducing financial losses and enhancing customer trust.

## 2. Data & Features
The model was developed using a dataset containing 1,000 card transaction records, with 700 used for training and 300 for testing. The dataset has a positive fraud rate of 7.6%. The features utilized in the model include:

- **amount**: The transaction amount.
- **merchant_risk**: A risk score associated with the merchant.
- **hour**: The hour of the transaction.
- **velocity_24h**: The number of transactions made by the cardholder in the last 24 hours.
- **is_foreign**: A binary indicator of whether the transaction is international.
